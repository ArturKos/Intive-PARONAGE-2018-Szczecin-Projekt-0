# Intive Patronage 2018 — Task 0

Android TODO application with splash screen, developed as part of the Intive Patronage 2018 program (Szczecin, Java track).

## Features

- SQLite database with ContentProvider for data access
- CursorLoader for asynchronous, lifecycle-aware data loading
- Multi-activity architecture: splash screen, task list, detail/edit screens
- Create, edit, and delete tasks with categories

## Tech Stack

- Java, Android SDK (API 21–26)
- Room Persistence Library
- Android Architecture Components
